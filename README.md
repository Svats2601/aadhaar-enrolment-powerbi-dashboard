# Aadhaar Enrolment Analysis Dashboard | Power BI

## Project Overview
This project analyzes Aadhaar enrolment data published by UIDAI using Power BI.
The dashboard provides insights into enrolment trends across states, districts, and age groups in India.

## Dashboard Highlights
- Total enrolments: **5.44+ million**
- State-wise and district-wise performance analysis
- Monthly enrolment trend analysis
- Demographic analysis by age group (0–5, 5–17, 18+)

## Project Structure
- **data/** – Raw UIDAI Excel datasets  
- **dashboard/** – Power BI (.pbix) dashboard file  
- **reports/** – Exported PDF dashboard  

## Tools & Technologies
- Power BI
- DAX
- Excel
- Data Modeling
- Data Cleaning
- Data Visualization

## Data Source
- UIDAI (Unique Identification Authority of India) – Official Aadhaar Enrolment Data

## Key Insights
- Uttar Pradesh leads Aadhaar enrolments
- Child (0–5) registrations form the largest share
- Top 5 states contribute the majority of enrolments
- Noticeable district-level variation across regions
